Java注解 和反射

1. 元数据

元数据是描述数据的数据

怎样定义注解?

自定义注解上可以使用元注解@Documented,@Target, @Retention等

@Target({ElementType.METHOD}) 表示自定义的注解只能用在方法上

@Retention(RetentionPolicy.SOURCE)

RetentionPolicy.SOURCE 表示注解只存在于源码中

RetentionPolicy.CLASS 表示注解存在于class文件中

RetentionPolicy.RUNTIME

Java中的注解其实是一种配置

2 . Java 反射

Class 类, 是“一面镜子”能照出类的信息: 类所属包，类名，字段名和字段类型等

任何一个类都有且仅有一个Class对象

怎样获取一个类中的Class对象?

a) 对象.getClass()

b) 类.class

c) Class.forName(“类的全路径”)

JS中函数调用： 函数名(), 函数名.call() 函数名.apply()

使用反射创建对象 Class

使用clone创建对象 类必须实现Cloneable接口

使用 序列化和反序列化创建对象 必须实现Serializable接口

序列化： 将对象写到 文件中

反序列化: 将文件读取出来转换成对象
